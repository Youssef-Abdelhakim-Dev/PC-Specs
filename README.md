# Full Technical PC Specs - PowerShell Script

## Overview

This PowerShell script generates a **comprehensive HTML report of your PC** with detailed technical specifications.  
It is designed for **web developers, cybersecurity professionals, and IT admins** who want a complete snapshot of a system's hardware, software, and configuration.

The report includes information on:

- Operating System
- CPU
- RAM
- GPU
- Disk drives and partitions
- Motherboard & BIOS
- Network adapters
- Firewall status
- Installed antivirus
- Installed software
- PowerShell modules
- Environment variables

The output is a **well-formatted HTML file**, which can be easily shared or archived.

---

## Features

- Retrieves **system and OS details** (version, build, architecture, registered user)
- Detects **CPU and RAM specifications**
- Lists **GPU details** including VRAM
- Detects **disk drives and partitions** with free space
- Retrieves **motherboard and BIOS information**
- Monitors **network adapters and firewall status**
- Checks installed **antivirus products**
- Lists all **installed software** (useful for devs and cybersecurity)
- Captures **PowerShell modules** and **environment variables**
- Generates a **color-coded HTML report** with tables for easy reading

---

## Requirements

- Windows OS
- PowerShell 5.1 or higher
- Read/write permissions for the output folder
